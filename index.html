Sonar Glasses

A New Vision for the Blind

Empowering the visually impaired with real-time obstacle detection and spatial awareness through acoustic imaging.

What is Sonar Glasses?

Sonar Glasses are an innovative wearable technology designed to assist individuals with visual impairments. By using advanced sensors to "see" the environment, the glasses translate spatial information into intuitive audio feedback, helping users navigate the world with greater ease and safety.

How It Works

  • Sound waves emitted: The device sends out high-frequency ultrasonic pulses into the environment.
  • Echo reflected: These waves bounce off nearby objects and return to the sensors.
  • Distance calculated: An onboard processor measures the time-of-flight to determine the exact distance of obstacles.
  • Audio feedback to user: The system generates varying audio tones or haptic pulses to inform the user about the object's proximity.

Components Used

  • Ultrasonic sensors (HC-SR04 or similar)
  • LiDAR (for high-precision mapping)
  • Audio processor (Microcontroller/DSP)
  • Rechargeable Lithium-ion Battery
  • Bone conduction feedback system

Benefits

  • Safe navigation: Detects head-level obstacles that traditional canes might miss.
  • Independence: Reduces reliance on sighted guides for daily movement.
  • Confidence: Provides a mental map of the surroundings, increasing user comfort.
  • Real-time awareness: Instant feedback allows for fluid movement in dynamic environments.

Who Can Use This

  • Blind individuals
  • Visually impaired users
  • Senior citizens with age-related vision loss

Why This Is Different

Traditional aids like the white cane only detect ground-level obstacles, while guide dogs require significant maintenance and training. Sonar Glasses offer a hands-free, tech-driven solution that provides upper-body protection and environmental awareness without the physical limitations of a cane or the responsibility of a live animal.

Future Scope

  • AI integration: Object recognition (e.g., "Door ahead", "Chair nearby").
  • Smaller design: Miniaturization into standard eyeglass frames.
  • Affordable pricing: Mass production to make it accessible in developing regions.
  • Wider accessibility: Integration with GPS for turn-by-turn navigation.

FAQ

Is it safe?
Yes, it uses low-power ultrasonic waves similar to those used in parking sensors, which are completely harmless to humans and pets.
Does it work indoors?
Absolutely. It is highly effective in confined spaces, helping users navigate corridors and avoid furniture.
Is it affordable?
The current prototype is built with cost-effective components. Our goal is to keep the final product much cheaper than existing high-tech aids.
Can it replace a cane?
While it provides significant advantages, we recommend using it as a complementary tool to the white cane for maximum safety.
Is this available in the market?
Currently, it is a student innovation project in the prototype stage. We are looking for partners to bring it to market.
